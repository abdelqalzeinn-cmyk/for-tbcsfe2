# bcsfe_rs

A rust port of <https://github.com/fieryhenry/BCSFE-Python>.

## Build from source

```sh
git clone https://codeberg.org/fieryhenry/bcsfe_rs.git
cd bcsfe_rs
cargo run --features gui
```
